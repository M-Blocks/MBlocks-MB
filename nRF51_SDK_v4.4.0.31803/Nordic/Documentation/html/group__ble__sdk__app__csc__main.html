Detailed Description

This file contains the source code for a sample application using the Cycling Speed and Cadence Service. It also includes the sample code for Battery and Device Information services. This application uses the srvlib_conn_params module.

This application implements supports for both Wheel revolution Data and Crank Revolution Data. In addition, this application also has support for all 'Speed and Cadence Control Point'.
